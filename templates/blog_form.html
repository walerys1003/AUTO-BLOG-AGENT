Poradnik API WordPress

Aby połączyć blog WordPress, potrzebujesz:

  1. Dostęp do REST API: REST API musi być włączone (domyślnie w WordPress 4.7+).
  2. Hasło aplikacji: Wygeneruj je w panelu admina → Użytkownicy → Twój profil.
  3. Odpowiednie uprawnienia: Użytkownik musi mieć prawo tworzenia i edycji postów.
Nigdy nie udostępniaj swojego tokenu API ani hasła aplikacji innym osobom.

Format kategorii — przykład

Wprowadź kategorie jako tablicę JSON obiektów z polami id i name.

[
  { "id": 1, "name": "Technologia" },
  { "id": 5, "name": "Zdrowie" },
  { "id": 8, "name": "Podróże" }
]

ID kategorii znajdziesz w panelu administracyjnym WordPress: Wpisy → Kategorie. ID jest zwykle widoczne w adresie URL podczas edycji kategorii.
