Social Media Setup Guide

To automatically share your blog content on social media, you'll need to set up API access for each platform:

Facebook
  1. Create a Facebook Developer account
  2. Create a new app (Business type)
  3. Add the "Facebook Login" product
  4. Generate a Page Access Token for your Facebook page
  5. Use your Page ID as the Account ID and the token as API Token when adding the account
Twitter
  1. Create a Twitter Developer account
  2. Create a new project and app
  3. Apply for Elevated access to use write capabilities
  4. Generate API Key and API Secret
  5. Set API Key as API Token and API Secret as API Secret when adding the account
LinkedIn
  1. Create a LinkedIn Developer account
  2. Create a new app
  3. Request the "Share on LinkedIn" and "Sign In with LinkedIn" permissions
  4. Generate a Client ID and Client Secret
  5. Use Client ID as API Token, Client Secret as API Secret, and your LinkedIn user ID as Account ID
Instagram
  1. Create a Facebook Developer account
  2. Create a new app (Business type)
  3. Add the "Instagram Graph API" product
  4. Connect to your Instagram Business Account
  5. Generate an access token for your Instagram account
  6. Use your Instagram Business ID as Account ID and the token as API Token
Each social media platform requires different API access setup. Refer to each platform's developer documentation for the most up-to-date instructions.
