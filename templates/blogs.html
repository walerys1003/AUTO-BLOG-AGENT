WordPress Configuration Guide

To connect your WordPress blog to BlogAutomationAgent, you'll need the following:

  1. WordPress REST API Access: Ensure your WordPress installation has the REST API enabled (it's enabled by default in WordPress 4.7+).
  2. Application Password: Generate an application password in WordPress:
    • Go to your WordPress admin panel → Users → Your Profile
    • Scroll down to "Application Passwords" section
    • Enter a name for the application (e.g., "BlogAutomationAgent")
    • Click "Add New Application Password"
    • Copy the generated password (you'll need it when adding the blog)
  3. Categories: Prepare a list of category IDs you want to publish to. You can find these in your WordPress admin panel under Posts → Categories.
For enhanced security, we recommend using HTTPS for your WordPress site and creating a dedicated user with Editor permissions for API access.
