Jak podłączyć blog WordPress

Aby połączyć swojego bloga WordPress z MasterContentAI, potrzebujesz trzech rzeczy:

  1. Włączone REST API: Twoja instalacja WordPress musi mieć aktywne REST API (jest domyślnie włączone od wersji 4.7).
  2. Hasło aplikacji: wygeneruj je w WordPress:
    • panel administratora → Użytkownicy → Twój profil,
    • sekcja Hasła aplikacji,
    • wpisz nazwę (np. MasterContentAI) i kliknij Dodaj nowe hasło aplikacji,
    • skopiuj wygenerowane hasło — wpiszesz je przy dodawaniu bloga.
  3. Kategorie: przygotuj listę identyfikatorów kategorii, do których chcesz publikować (znajdziesz je w Wpisy → Kategorie).
Dla bezpieczeństwa zalecamy używanie HTTPS oraz utworzenie dedykowanego użytkownika z uprawnieniami Redaktora wyłącznie do dostępu przez API.
